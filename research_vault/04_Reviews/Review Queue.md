# Review Queue

Role-based review is the filter layer between raw research and useful signal.

## Active Roles

- `CEO` - strategic relevance, market direction, partnership fit, narrative weight
- `CTO` - technical implications, tooling signals, implementation relevance, architecture patterns

## Review Flow

1. Capture or import notes into `00_Inbox/`.
2. Normalize high-signal items.
3. Review by role.
4. Publish the useful signal into the latest reports.
5. Promote only what deserves governed follow-through.
