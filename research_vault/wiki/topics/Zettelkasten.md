# Zettelkasten

A method of note-taking and knowledge management using atomic, interlinked notes.

## Key Principles

- **Atomic notes** — one idea per note
- **Stable IDs** — each note has a permanent identifier
- **Immutable** — notes are never modified, only linked
- **Emergent structure** — structure emerges from links, not folders

## Comparison to LLM Wiki

- Zettelkasten is human-curated; LLM Wiki is AI-maintained
- Both use immutable notes and backlinks
- Zettelkasten emphasizes human judgment; LLM Wiki scales maintenance
