# Agentic Architecture

The design of systems where AI agents plan, execute, and coordinate work.

## Key Concepts

- **Conductor** — plans, decomposes, delegates
- **Workers** — execute scoped tasks
- **Orchestration layer** — communication hub for agents
- **Think/Do model** — think layer plans, do layer executes
