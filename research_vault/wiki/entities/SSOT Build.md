# SSOT Build

Single Source of Truth build project. The canonical authoring surface for horse and lease data.

## Role

- Producer-side truth for horse and listing data
- HLT (Horse, Lease, Terms) assembly
- Firestore-backed persistence

## Key Concepts

- Horse identity truth (microchip + Stud Book evidence)
- Current association readiness (trainer/stable, owner, governing body)
- Lease qualification
