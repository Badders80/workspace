# OpenFang Memory Strategy

This folder is for documenting memory design, retention policies, and ingestion strategy.

What belongs here:
- memory architecture notes
- hand memory policies
- guidance for how to use the OpenFang knowledge graph
- a map of what gets persisted vs what is ephemeral

What does not belong here:
- the actual OpenFang memory store
- runtime data files

Actual OpenFang memory lives in `~/.openfang/data` and is managed by OpenFang itself.
