# Magic Animator — AI Animation Export

## Source
- Website: Magic Animator (creator tool)
- Type: AI animation export flow
- Input: Figma or Canva assets

## Purpose
Export AI-generated animations from Figma/Canva directly into video projects. Ideal for creators who design in Figma but need programmatic motion for reels.

## Setup

```bash
# Check if npm package exists
npm search magic-animator

# Or use web interface
# Visit magicanimator.design
```

## How It Works

1. Design assets in Figma or Canva
2. Export to Magic Animator
3. AI applies motion, easing, transitions
4. Download as video/GIF or send to Remotion/FFmpeg

## For Faceless Reels
Magic Animator is useful for:
- Animated brand elements (logos, badges)
- Smooth transitions between scenes
- B-roll motion effects
- Text animation presets

## Comparison: Magic Animator vs Remotion

| Feature | Magic Animator | Remotion |
|---------|-----------------|----------|
| Design Source | Figma/Canva | Code |
| Learning Curve | Low | Medium |
| Precision | Medium | High |
| Automation | GUI-based | Prompt-based |
| Best For | Designers | Developers |
