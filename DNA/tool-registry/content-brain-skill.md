# Content Brain — Feedback Looping System

## Source
- Instagram: @charlieautomates
- Pipeline: Go Viral Bro + Remotion Agent Skills
- Creator: Charles Dove

## Purpose
A complete feedback-looping content system that analyzes performance, surfaces viral patterns, generates better inputs, and produces reusable assets. The "brain" learns from your data over time.

## Architecture

```
Research → Scripting → Visual Production → Posting → Analysis → Loop
    ↑                                                           |
    └─────────────────────── Feedback ─────────────────────────┘
```

## Components

### 1. Research (Go Viral Bro)
- Scrapes competitors (YouTube/Instagram)
- Keyword searches (YouTube/Reddit/GitHub)
- Transcribes winning content
- Extracts hooks

### 2. Scripting (Go Viral Bro)
- Contrast Formula generates angles
- Hook scoring (6 patterns, rated 1-10)
- Full scripts with filming cards
- PDF lead magnets

### 3. Visual Production (Remotion)
- Programmatic video rendering
- B-roll, captions, animations
- Brand-compliant output

### 4. Analysis
- Pulls analytics from platforms
- Identifies performance patterns
- Updates brain stores (JSONL)

## For Faceless Reels

The Content Brain is ideal for faceless channels because:
- **No face required** — text/graphic/voiceover outputs
- **Data-driven hooks** — 6 proven patterns, scored
- **Feedback loop** — improves over time from your performance
- **Scale** — research once, generate many variants

## Evolution_Content Integration

1. **Brain input** → Tournament/race data (what pundits predicted)
2. **Brain output** → Script with hooks + filming cards
3. **Remotion render** → Kingmaker template video
4. **Post** → Instagram + X
5. **Analyze** → Engagement metrics → update brain

## JSONL Stores
The brain maintains:
- `topics.jsonl` — successful content themes
- `angles.jsonl` — winning angles
- `hooks.jsonl` — proven hooks
- `scripts.jsonl` — past scripts
- `insights.jsonl` — performance learnings
