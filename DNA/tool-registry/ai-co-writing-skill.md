## Purpose
Specialized prompt files for iterative co-writing with Claude. Handles tone-matching, framework application, and multi-pass editing. Turns Claude into a creative writing partner that matches your voice.

## Setup

```bash
git clone https://github.com/az9713/ai-co-writing-claude-skills.git

```

## Key Capabilities

| Capability | What it does |
|------------|--------------|
| **Tone-Matching** | Learns your writing style from examples, produces consistent voice |
| **Framework Application** | Applies proven structures (hook-story-CTA, AIDA, PAS) automatically |
| **Multi-Pass Editing** | Iterates through multiple editing passes: structure → flow → polish |
| **A/B Variants** | Generates multiple script variations for testing |

## How It Works

1. Provide writing samples (your style)
2. Input a rough script or idea
3. Claude applies: tone-matching → framework → multi-pass editing
4. Output: polished, on-brand scripts
