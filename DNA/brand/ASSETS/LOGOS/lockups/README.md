# Evolution Stables Logo Lockups

This directory contains SVG templates and guidelines for creating Evolution Stables logo lockups (combinations of wordmark and monogram).

## Source Assets
All lockups should be created using these source files:
- **Wordmark**: `/primary/Evolution-Stables-Logo-*.svg` (White or Black versions)
- **Monogram**: `/marks/EvolutionStables Mono-*.svg` (Gold, White, or Black versions)

## Lockup Variations

### 1. Horizontal Lockup
- **Usage**: Standard application, website headers, documents
- **Composition**: Monogram (left) + Wordmark (right)
- **Spacing**: 40px between monogram and wordmark
- **Vertical Alignment**: Center-aligned
- **Clear Space**: Equal to the height of the monogram

### 2. Vertical Lockup  
- **Usage**: Narrow applications, social media profiles, mobile apps
- **Composition**: Wordmark (top) + Monogram (bottom)
- **Spacing**: 20px between elements
- **Horizontal Alignment**: Center-aligned
- **Clear Space**: Equal to the width of the widest element

### 3. Stacked Lockup
- **Usage**: Very narrow applications, app icons, favicons
- **Composition**: Monogram (top) + Wordmark (bottom, two-line)
- **Spacing**: 10px between elements
- **Horizontal Alignment**: Center-aligned
- **Wordmark**: Split into two lines for better proportions

## Color Variations

### Full Color (Recommended)
- **Background**: Transparent or `#09090b` (Velvet Night)
- **Monogram**: `#d4a964` (Brand Gold)
- **Wordmark**: `#f5f5f5` (Foreground) on dark backgrounds
- **Wordmark**: `#0a0a0a` (Surface) on light backgrounds

### Reverse
- **Background**: `#09090b` (Velvet Night)
- **Monogram**: `#d4a964` (Brand Gold)
- **Wordmark**: `#f5f5f5` (Foreground)

### Monochrome
- **Background**: Transparent or appropriate background color
- **Monogram**: `#0a0a0a` (Surface) or `#f5f5f5` (Foreground)
- **Wordmark**: Opposite value for contrast

## Production Guidelines

### Using Vector Tools (Illustrator, Figma, Sketch):
1. Open the appropriate wordmark SVG (white or black version)
2. Open the appropriate monogram SVG (gold, white, or black version)
3. Position elements according to the lockup type specifications
4. Ensure proper spacing and alignment
5. Group elements and export as needed

### Export Formats:
- **AI**: Source files for print production
- **SVG**: Web and scalable applications  
- **PNG**: Digital applications with transparency (multiple sizes: 24px, 32px, 48px, 64px, 128px, 256px)
- **JPG**: Digital applications without transparency needs
- **EPS**: Legacy print workflows

### Size Guidelines:
- **Minimum Width**: 120px for horizontal lockups
- **Minimum Height**: 40px for horizontal lockups
- **Clear Space**: Always maintain minimum clear space equal to the height of the monogram mark

## File Naming Convention:
```
evolution-stables-lockup-[type]-[variation].[format]
```
Examples:
- `evolution-stables-lockup-horizontal-fullcolor.svg`
- `evolution-stables-lockup-vertical-reverse.png`
- `evolution-stables-lockup-stacked-monochrome.jpg`

## Applications:
- **Horizontal**: Website headers, letterhead, documents, standard applications
- **Vertical**: Social media profiles, mobile apps, narrow vertical spaces
- **Stacked**: App icons, favicons, very narrow applications
